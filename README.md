# Calculator

A modern, user-friendly calculator web application with dark/light theme support. Built with vanilla JavaScript, this calculator provides a seamless calculation experience with a beautiful, responsive interface.

## Project Function

This calculator performs basic and advanced arithmetic calculations with a clean, intuitive interface. It supports:

- Basic operations: Addition, subtraction, multiplication, and division
- Advanced functions: Percentage calculation, digit deletion, and clear all
- Real-time result preview and visual feedback
- Light and dark theme toggle for comfortable viewing
- Responsive design that works on all devices
- Keyboard support for efficient number entry

## How to Run

### Quick Start

1. Open the `calculator` folder
2. Double-click on `index.html` or right-click and select "Open with" your preferred web browser
3. The calculator will load immediately in your browser

**No installation or server required** - it runs entirely in your browser!
